# Rafhael Marsigli

> Personal Website & Blog

This is the repository for my personal website, portfolio, and blog. Built with a focus on **high performance**, **technical SEO**, and **modern design**, serving as the central hub for my articles, open-source projects, and consulting services.

The site was developed to be blazing fast and accessible, reflecting modern web engineering best practices. It consumes data from a headless CMS (Rush CMS) and is fully bilingual (English and Portuguese).

## Key Highlights

- **Tech Blog:** Articles covering software engineering, architecture, Artificial Intelligence, and career growth.
- **Open Source:** Showcase of personal projects and tools built for the community.
- **Internationalization (i18n):** Complete content structure available in both EN and PT-BR.
- **Sharp SEO:** Metadata, schema markup, dynamic sitemap, and precise indexation control tailored for top Google performance.

## Under the Hood

Instead of overcomplicating things, the stack is modern, straightforward, and built for speed:

- **[Astro](https://astro.build/)**: For static site generation and ultra-fast HTML delivery.
- **[Tailwind CSS v4](https://tailwindcss.com/)**: For frictionless styling.
- **Rush CMS**: Headless CMS for managing content.
- **Expressive Code**: For beautiful, syntax-highlighted code blocks in the blog.
